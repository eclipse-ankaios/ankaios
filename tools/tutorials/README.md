# Tutorials

This directory contains subfolders with examples of real scenarios using Eclipse Ankaios as a software orchestrator.

## vehicle_signals

In this tutorial, we will show you how to use Ankaios to set up workloads that publish and subscribe to vehicle signals.
The full documentation is available here: [Sending and receiving vehicle signals](https://eclipse-ankaios.github.io/ankaios/main/usage/tutorial-vehicle-signals/).

## fleet_management

This tutorial will show you how to manage a fleet of vehicles running Ankaios.
The full documentation is available here: [Manage a fleet of vehicles from the cloud](https://eclipse-ankaios.github.io/ankaios/main/usage/tutorial-fleet-management/).
